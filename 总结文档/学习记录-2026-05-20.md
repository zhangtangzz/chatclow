## 📖 今天学到的知识（2026-05-20）

### 1. Spring Boot 项目结构

```
chatclow_api/
├── pom.xml                          ← Maven 项目配置
├── src/main/java/com/chatclow/
│   ├── ChatclowApplication.java     ← 启动类（入口）
│   ├── entity/                     ← 实体类（对应数据库表）
│   ├── mapper/                     ← 数据库操作层
│   ├── service/                    ← 业务逻辑层
│   │   ├── UserService.java        ← 服务接口
│   │   └── impl/UserServiceImpl.java ← 服务实现
│   └── controller/                 ← 控制器（API 接口）
│       └── UserController.java
└── src/main/resources/
    └── application.yml              ← 配置文件
```

### 2. 分层架构（重要！）

```
浏览器请求
    ↓
Controller（控制器）← 接收请求，返回数据
    ↓
Service（服务层）← 写业务逻辑
    ↓
Mapper（映射层）← 操作数据库
    ↓
数据库
```

**类比**：餐厅点餐
- Controller = 服务员（接收点单）
- Service = 厨师（做菜）
- Mapper = 冰箱管理员（拿食材）

### 3. 关键注解

| 注解 | 作用 | 使用位置 |
|------|------|---------|
| `@SpringBootApplication` | 标记启动类 | ChatclowApplication.java |
| `@RestController` | 标记控制器，返回 JSON | UserController.java |
| `@RequestMapping("/api/user")` | 定义路径前缀 | UserController.java |
| `@GetMapping("/{username}")` | 处理 GET 请求 | UserController.java |
| `@PostMapping("/register")` | 处理 POST 请求 | UserController.java |
| `@Autowired` | 自动注入依赖 | UserServiceImpl.java, UserController.java |
| `@Service` | 标记服务类 | UserServiceImpl.java |
| `@MapperScan("com.chatclow.mapper")` | 扫描 Mapper 接口 | ChatclowApplication.java |
| `@TableName("chatclow_ai_user")` | 指定表名 | User.java |
| `@TableId(type = IdType.AUTO)` | 标记主键自增 | User.java |

### 4. 遇到的错误及解决方法

#### 错误1：找不到符号 `RequestMapping`
**原因**：缺少 import 语句  
**解决**：添加 `import org.springframework.web.bind.annotation.RequestMapping;`

#### 错误2：QueryWrapper 报错
**原因**：大小写错误，`QueryWrapper<user>` 应该是 `QueryWrapper<User>`  
**解决**：Java 区分大小写，改成大写 `User`

#### 错误3：启动报错 `No qualifying bean of type 'UserMapper'`
**原因**：MyBatis 没有扫描到 Mapper  
**解决**：在启动类添加 `@MapperScan("com.chatclow.mapper")`

#### 错误4：404 Not Found
**原因**：`@RequestMapping` 没有指定路径  
**解决**：改成 `@RequestMapping("/api/user")`

#### 错误5：SQL 错误 `Unknown column 'update_dt'`
**原因**：User.java 有 `updateDt` 字段，但数据库表没有这列  
**解决**：删除 User.java 中的 `updateDt` 字段

## 📋 当前项目代码详解

### 1. ChatclowApplication.java（启动类）

```java
package com.chatclow;

import org.mybatis.spring.annotation.MapperScan;
import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
@MapperScan("com.chatclow.mapper")  // 扫描 Mapper 接口
public class ChatclowApplication {
    public static void main(String[] args) {
        SpringApplication.run(ChatclowApplication.class, args);
    }
}
```

**关键点**：
- `@SpringBootApplication`：标记这是 Spring Boot 应用
- `@MapperScan("com.chatclow.mapper")`：告诉 Spring 去哪里找 Mapper

### 2. User.java（实体类）

```java
package com.chatclow.entity;

import com.baomidou.mybatisplus.annotation.*;
import lombok.Data;
import java.time.LocalDateTime;

@Data
@TableName("chatclow_ai_user")
public class User {
    @TableId(type = IdType.AUTO)
    private Long id;
    
    private String username;
    private String email;
    private String password;
    private Integer role;
    private Integer totals;
    
    @TableField(fill = FieldFill.INSERT)
    private LocalDateTime createdDt;
}
```

**关键点**：
- `@Data`：Lombok 自动生成 getter/setter
- `@TableName("chatclow_ai_user")`：对应数据库表名
- `@TableId(type = IdType.AUTO)`：主键自增

### 5. UserServiceImpl.java（服务实现）

```java
package com.chatclow.service.impl;

import com.baomidou.mybatisplus.core.conditions.query.QueryWrapper;
import com.baomidou.mybatisplus.extension.service.impl.ServiceImpl;
import com.chatclow.entity.User;
import com.chatclow.mapper.UserMapper;
import com.chatclow.service.UserService;
import org.springframework.stereotype.Service;

@Service
public class UserServiceImpl extends ServiceImpl<UserMapper, User> implements UserService {
    
    @Override
    public User getByUsername(String username) {
        QueryWrapper<User> wrapper = new QueryWrapper<>();
        wrapper.eq("username", username);
        return baseMapper.selectOne(wrapper);
    }
    
    @Override
    public boolean register(String username, String password, String email) {
        User existUser = getByUsername(username);
        if (existUser != null) {
            return false;  // 用户名已存在
        }
        
        User user = new User();
        user.setUsername(username);
        user.setPassword(password);
        user.setEmail(email);
        user.setRole(1);
        user.setTotals(100);
        
        return this.save(user);
    }
}
```

**关键点**：
- `QueryWrapper`：条件构造器，相当于 SQL 的 `WHERE` 条件
- `baseMapper.selectOne(wrapper)`：查询一条记录

### 6. UserController.java（控制器）

```java
package com.chatclow.controller;

import com.chatclow.entity.User;
import com.chatclow.service.UserService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;

import java.util.HashMap;
import java.util.Map;

@RestController
@RequestMapping("/api/user")
public class UserController {
    
    @Autowired
    private UserService userService;
    
    @PostMapping("/register")
    public Map<String, Object> register(@RequestParam String username,
                                       @RequestParam String password,
                                       @RequestParam String email) {
        Map<String, Object> result = new HashMap<>();
        boolean success = userService.register(username, password, email);
        
        if (success) {
            result.put("code", 200);
            result.put("message", "注册成功！");
        } else {
            result.put("code", 500);
            result.put("message", "用户名已存在！");
        }
        return result;
    }
    
    @GetMapping("/{username}")
    public Map<String, Object> getUser(@PathVariable String username) {
        Map<String, Object> result = new HashMap<>();
        User user = userService.getByUsername(username);
        
        if (user != null) {
            result.put("code", 200);
            result.put("data", user);
        } else {
            result.put("code", 404);
            result.put("message", "用户不存在！");
        }
        return result;
    }
}
```

**关键点**：
- `@RequestParam`：获取请求参数（?username=xxx）
- `@PathVariable`：获取路径参数（/api/user/xxx）

## 🔑 关键概念总结

### 1. Spring Boot 核心概念
- **启动类**：包含 `main()` 方法，运行后启动整个应用
- **自动配置**：Spring Boot 自动配置很多东西，减少手动配置
- **依赖注入**：`@Autowired` 自动把需要的对象注入进来

### 2. MyBatis-Plus 核心概念
- **BaseMapper**：提供基本 CRUD 方法
- **IService**：提供更多服务层方法
- **QueryWrapper**：条件构造器，用来拼 SQL 条件

### 3. RESTful API 设计
- **GET**：查询资源
- **POST**：创建资源
- **PUT**：更新资源
- **DELETE**：删除资源
